IDLES Band Website

Project Info:

Solo project developed to allow users to explore the band IDLES. The user can choose one of the band's albums fro the homepage to be redirected to a page about that album. The album page displays a summary of the album, a tracklist from spotify with song samples, and the music videos associated with the selected album. The user can leave reviews on the review page and multiple users can chat with eachother on the chat page.

Project Details:

  • Client:DigitalCrafts
  • Industry:Music
  • Technologies:HTML5, CSS3, Bootstrap, JavaScript, ExpressJS, Sequelize, PostgreSQL, Socket.io
  • Date:July 2022
  • URL:idles-band.herokuapp.com
